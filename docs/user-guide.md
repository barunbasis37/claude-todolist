# User Guide

How to use the TodoList app once it's running (see [Getting Started](getting-started.md)
to launch it).

## Viewing your tasks

Open the app in your browser (`http://localhost:5232` by default). The home
page lists all your tasks, with incomplete tasks shown first and, within each
group, the newest tasks at the top.

## Adding a task

1. Type a title into the input field at the top of the list (up to 200
   characters).
2. Click **Add** (or press Enter).

The task appears at the top of the incomplete list.

## Marking a task complete

Click the checkbox next to a task to toggle it between complete and
incomplete. Completed tasks move down below any remaining incomplete tasks.

## Deleting a task

Click the **Delete** button next to a task to remove it permanently. This
cannot be undone.

## Other pages

- **Privacy** — a static placeholder page, linked from the site footer/nav.
- **Error** — shown automatically if something goes wrong while handling a
  request; displays a request ID for troubleshooting.
